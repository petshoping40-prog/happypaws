# Бот для формирования отчетов ИТ-аудита

Этот бот принимает фото (скриншоты ошибок, фото оборудования) и текстовое описание проблем, 
а затем с помощью ИИ (OpenAI GPT-4o) формирует профессиональный структурированный отчет.

## Установка

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Получите токены:**
   - **Telegram Token:** Создайте бота в @BotFather и получите токен.
   - **OpenAI API Key:** Получите ключ на платформе OpenAI (нужен доступ к моделям с поддержкой зрения, например, `gpt-4o`).

3. **Настройте переменные окружения:**
   
   Вы можете установить их прямо в терминале перед запуском:
   ```bash
   export TELEGRAM_TOKEN="ваш_телеграм_токен"
   export OPENAI_API_KEY="ваш_openai_ключ"
   ```
   
   Или просто отредактируйте файл `main.py`, вставив ключи в переменные `TELEGRAM_TOKEN` и `OPENAI_API_KEY`.

## Запуск

```bash
python main.py
```

## Как это работает

1. Пользователь отправляет боту фото и/или текст с описанием проблемы.
2. Бот отправляет данные в OpenAI API с системным промптом "ИТ-аудитор".
3. Нейросеть анализирует изображение (распознает текст ошибки, визуальные дефекты) и контекст.
4. Бот возвращает готовый отчет в формате:
   - Краткое резюме
   - Категория инцидента
   - Уровень критичности
   - Детальное техническое описание
   - Анализ визуальных данных
   - Предполагаемая причина
   - Рекомендации по решению

> 3. **Уровень критичности:** Критический
