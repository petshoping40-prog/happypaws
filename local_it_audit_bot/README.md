# Инструкция по установке и запуску локального ИТ-аудит бота

## 📋 Требования
- ПК с 16 ядрами CPU и 16 ГБ ОЗУ (Windows/Linux/Mac)
- Python 3.8+
- Telegram токен от @BotFather
- ~10 ГБ свободного места на диске

---

## 🚀 Шаг 1: Установка Ollama

### Для Windows:
1. Скачайте установщик с https://ollama.com/download/windows
2. Запустите установщик
3. После установки Ollama запустится автоматически

### Для Linux:
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### Для macOS:
```bash
brew install ollama
```

---

## 🤖 Шаг 2: Установка модели с поддержкой изображений

Откройте терминал/командную строку и выполните:

```bash
# Рекомендуемая модель (хороший баланс скорости и качества)
ollama pull llava

# Альтернативные модели (можно попробовать):
# ollama pull llava-llama3    # Более точная, но медленнее
# ollama pull moondream       # Очень быстрая, меньше точность
# ollama pull bakllava        # Альтернативная версия
```

Проверка установки:
```bash
ollama list
```

---

## 🐍 Шаг 3: Установка зависимостей Python

```bash
cd local_it_audit_bot
pip install -r requirements.txt
```

---

## ⚙️ Шаг 4: Настройка бота

1. Откройте файл `main.py`
2. Замените `ВАШ_ТОКЕН_БОТА` на токен от @BotFather
3. При необходимости измените `MODEL_NAME` (должно совпадать с установленной моделью)

```python
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"  # Ваш токен
MODEL_NAME = "llava"  # Должно совпадать с установленной моделью
```

---

## ▶️ Шаг 5: Запуск

### Терминал 1: Убедитесь что Ollama работает
Ollama обычно запускается как сервис автоматически. Проверьте:
```bash
# Windows: проверьте в диспетчере задач процесс ollama.exe
# Linux: 
systemctl status ollama

# Или просто попробуйте:
ollama list
```

Если не работает, запустите вручную:
```bash
ollama serve
```

### Терминал 2: Запуск бота
```bash
cd local_it_audit_bot
python main.py
```

Вы должны увидеть:
```
🤖 Бот запущен с моделью: llava
📡 Ollama API: http://localhost:11434/api/generate
Ожидаю сообщения...
```

---

## 🌐 Шаг 6: Доступ из интернета (без статического IP)

### Вариант A: Cloudflare Tunnel (рекомендуется)

1. Установите cloudflared:
   - Windows: скачайте с https://github.com/cloudflare/cloudflared/releases
   - Linux: `wget https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64 && chmod +x cloudflared-linux-amd64`

2. Зарегистрируйтесь на https://dash.cloudflare.com/

3. Создайте туннель:
```bash
cloudflared tunnel --url http://localhost:11434
```

⚠️ **Важно**: Для работы бота нужен только исходящий доступ к Telegram API, Ollama остается локальным. Telegram сам свяжется с вашим ботом через свои серверы.

### Вариант B: Tailscale (для приватного доступа)

1. Установите Tailscale: https://tailscale.com/download
2. Войдите под своим аккаунтом
3. Теперь вы можете подключаться к ПК из любой точки мира через Tailscale сеть

## 🔧 Настройка производительности

Для ускорения работы на вашем железе можно настроить Ollama:

Создайте файл `~/.ollama/config.json` (Linux/Mac) или `%USERPROFILE%\.ollama\config.json` (Windows):

```json
{
  "num_thread": 16,
  "num_gpu": 0
}
```

Перезапустите Ollama после изменений.

## 🛠 Troubleshooting

### Ошибка "Connection refused"
- Убедитесь что Ollama запущен: `ollama list`
- Проверьте порт: `netstat -an | grep 11434`

### Модель отвечает медленно
- Используйте более легкую модель: `ollama pull moondream`
- Закройте другие приложения
- Уменьшите размер изображения перед отправкой

### Ошибка "model not found"
- Установите модель: `ollama pull llava`
- Проверьте название в `main.py`
